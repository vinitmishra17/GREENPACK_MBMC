
# GREENPACK_MBMC — Packaging Waste Awareness & Recycling Sign-Up

A prototype website for **GreenPack Recycling**, built for the MBMC area to raise awareness about packaging waste and enable residents to sign up for recycling pickup.

## Live Demo
https://vinitmishra17.github.io/GREENPACK_MBMC/

## Sections
1. Hero Introduction
2. Problem Description
3. Why It Matters
4. Recycling Sign-Up Form
5. Sustainable Living Tips
6. Footer

## Tech Stack
HTML, CSS, Vanilla JavaScript

## How to Run Locally
1. Clone:
 git clone https://vinitmishra17.github.io/GREENPACK_MBMC/
2. Open folder
3. Open `index.html` in your browser
